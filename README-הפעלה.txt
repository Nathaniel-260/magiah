מגיה - מנוע הגהה
=================

הפעלה מהירה
-----------
לחצו לחיצה כפולה על הקובץ:

    הפעלת מגיה.bat

ייפתח חלון שחור (חלון פקודה) ובתוך כמה שניות ייפתח מאליו הממשק בדפדפן.
אם שם הקובץ בעברית מקשה עליכם, השתמשו בקובץ הזהה:

    run-magiah.bat


מה קורה כשמפעילים
------------------
- נפתח שרת מקומי במחשב שלכם בלבד (הכתובת מתחילה ב-http://127.0.0.1).
- הדפדפן נפתח אוטומטית על הממשק.
- כדי לסגור: פשוט סגרו את החלון השחור.


היכן נשמרים הנתונים
--------------------
תוצאות הסריקה ובסיסי הנתונים נשמרים בתיקייה בשם:

    magiah_data

הנמצאת לצד קובץ ההפעלה (נוצרת אוטומטית בפעם הראשונה).

כדי להפנות את הממשק לתיקייה אחרת (למשל תיקייה שכבר יש בה report.db):
פתחו את קובץ ה-.bat בעזרת Notepad, וערכו את השורה המסומנת:

    set "MAGIAH_OUT=%~dp0magiah_data"

והחליפו אותה בנתיב המלא לתיקייה הרצויה, לדוגמה:

    set "MAGIAH_OUT=D:\scans\my_project"

שמרו וסגרו, והפעילו שוב את הקובץ.


דרישות
-------
יש להתקין Python בגרסה 3.9 ומעלה מהכתובת:

    https://www.python.org/downloads/

בזמן ההתקנה סמנו את האפשרות "Add Python to PATH".
אם Python אינו מותקן, קובץ ההפעלה יציג הודעה מתאימה ולא ייסגר, כדי שתספיקו לקרוא אותה.


פתרון תקלות
-----------
- אם החלון נסגר מיד: הפעילו שוב - הקובץ עוצר על הודעת שגיאה ומחכה ללחיצת Enter.
- אם הדפדפן לא נפתח: העתיקו לדפדפן ידנית את הכתובת המופיעה בחלון (למשל http://127.0.0.1:8766/).
- אם הפורט תפוס: מגיה בוחר אוטומטית פורט פנוי אחר - הכתובת בחלון תעודכן בהתאם.
